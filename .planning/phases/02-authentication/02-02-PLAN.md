<objective>
Build the Auth Validator sub-workflow in n8n that all webhook workflows will call as their first step to validate Supabase JWTs and extract user identity.

Purpose: This is the server-side security gate (INFRA-04). Every future webhook must validate the caller's identity before processing any request. Without this sub-workflow, webhooks remain open to unauthenticated access (CRIT-3). Building it as a reusable sub-workflow means each webhook adds one node instead of duplicating validation logic.

Output: A deployed n8n sub-workflow that accepts webhook request data, validates the JWT, and returns either `{ user_id, email }` on success or `{ error, statusCode: 401 }` on failure. Plus a test webhook that demonstrates correct integration.
</objective>

**Read 02-01-SUMMARY.md first** to determine:

**Sub-workflow design (two paths based on CORS_RESULT):**

**If CORS_RESULT = "header" (Authorization header approach):**

```
Node 1: Execute Sub-Workflow Trigger
  - Receives: { headers, body, query } from parent webhook

Node 2: Code - Extract Token
  - Input: headers from trigger
  - Logic:
    const authHeader = $input.first().json.headers?.authorization || $input.first().json.headers?.Authorization || '';
    if (!authHeader.startsWith('Bearer ')) {
      return [{ json: { authenticated: false, error: 'Missing or invalid Authorization header', statusCode: 401 } }];
    }
    const token = authHeader.replace('Bearer ', '');
    return [{ json: { token, body: $input.first().json.body, query: $input.first().json.query } }];

Node 3: IF - Check Token Exists
  - Condition: {{ $json.authenticated }} is not equal to false
  - True path -> Node 4
  - False path -> Node 6 (error output)

Node 4: JWT - Verify Token
  - Operation: Verify
  - Token: {{ $json.token }}
  - Credential: "Supabase JWT Auth" (ID from 02-01-SUMMARY.md)
  - On success -> Node 5
  - On error -> Node 6

Node 5: Code - Extract Claims and Validate
  - Logic:
    const payload = $input.first().json;
    const sub = payload.sub;
    const exp = payload.exp;
    const aud = payload.aud;

    // Check expiration
    if (exp && exp < Math.floor(Date.now() / 1000)) {
      return [{ json: { authenticated: false, error: 'Token expired', statusCode: 401 } }];
    }

    // Check audience
    if (aud !== 'authenticated') {
      return [{ json: { authenticated: false, error: 'Invalid token audience', statusCode: 401 } }];
    }

    if (!sub) {
      return [{ json: { authenticated: false, error: 'Token missing user ID', statusCode: 401 } }];
    }

    return [{ json: {
      authenticated: true,
      user_id: sub,
      email: payload.email || '',
      body: $('Code - Extract Token').first().json.body,
      query: $('Code - Extract Token').first().json.query
    }}];

Node 6: Code - Error Output
  - Logic:
    return [{ json: {
      authenticated: false,
      error: $input.first().json.error || 'JWT validation failed',
      statusCode: 401
    }}];
```

**If CORS_RESULT = "body" (token-in-body approach):**

Same structure but Node 2 changes to:

```javascript
Node 2: Code - Extract Token
  const body = $input.first().json.body || {};
  const token = body._auth_token;
  if (!token) {
    return [{ json: { authenticated: false, error: 'Missing _auth_token in request body', statusCode: 401 } }];
  }
  // Remove _auth_token from body before passing downstream
  const { _auth_token, ...cleanBody } = body;
  return [{ json: { token, body: cleanBody, query: $input.first().json.query } }];
```

**Alternative: If JWT node is unavailable or unreliable, use Supabase HTTP API validation instead of Node 4:**

```
Node 4: HTTP Request - Validate via Supabase
  - Method: GET
  - URL: https://llpnwaoxisfwptxvdfed.supabase.co/auth/v1/user
  - Headers:
    - Authorization: Bearer {{ $json.token }}
    - apikey: sb_publishable_9gnLO_J9HBxVta1f8Ecvxw_KO8huPYQ
  - On success (200) -> Node 5 (extract user from response body)
  - On error (401) -> Node 6
```

If using this approach, Node 5 becomes:
```javascript
const user = $input.first().json;
return [{ json: {
  authenticated: true,
  user_id: user.id,

  body: $('Code - Extract Token').first().json.body,
  query: $('Code - Extract Token').first().json.query
}}];
```

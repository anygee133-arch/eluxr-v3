**Test 2: Test CORS with Authorization header**

Create a minimal HTML file to test CORS from a browser context:

```bash
cat > /tmp/cors-test.html << 'HTMLEOF'
<!DOCTYPE html>
<html>
<head><title>CORS Test</title></head>
<body>
<h1>CORS Test for n8n Cloud + Authorization Header</h1>
<button onclick="testCORS()">Run CORS Test</button>
<pre id="output"></pre>
<script>
async function testCORS() {
  const output = document.getElementById('output');
  output.textContent = 'Testing...\n';

  // Test 1: POST with Authorization header (triggers preflight)
  try {
    const resp = await fetch('https://flowbound.app.n8n.cloud/webhook/eluxr-approval-list', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer test-token-12345'
      },
      body: JSON.stringify({ test: true })
    });
    output.textContent += `Test 1 (POST + Auth header): ${resp.status} ${resp.statusText}\n`;
    const data = await resp.text();
    output.textContent += `Response: ${data.substring(0, 200)}\n`;
    output.textContent += 'RESULT: CORS WORKS with Authorization header!\n\n';
  } catch (e) {
    output.textContent += `Test 1 FAILED: ${e.message}\n`;
    output.textContent += 'RESULT: CORS BLOCKS Authorization header.\n';
    output.textContent += 'FALLBACK: Will use token-in-body approach.\n\n';
  }

  // Test 2: POST without Authorization header (baseline)
  try {
    const resp2 = await fetch('https://flowbound.app.n8n.cloud/webhook/eluxr-approval-list', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ test: true })
    });
    output.textContent += `Test 2 (POST without Auth header): ${resp2.status}\n`;
    output.textContent += 'Baseline: POST without Auth header works.\n';
  } catch (e) {
    output.textContent += `Test 2 FAILED: ${e.message}\n`;
  }
}
</script>
</body>
</html>
HTMLEOF
```

Serve this file and test in a browser:

```bash

cd /tmp && python3 -m http.server 8888 &
# Then open http://localhost:8888/cors-test.html in browser
# Click "Run CORS Test" and record the results

# Option B: Use curl to simulate (does NOT test CORS, only connectivity)
# This is a backup to verify the endpoint responds:
curl -v -X OPTIONS "https://flowbound.app.n8n.cloud/webhook/eluxr-approval-list" \
  -H "Origin: http://localhost:8888" \
  -H "Access-Control-Request-Method: POST" \
  -H "Access-Control-Request-Headers: Content-Type, Authorization" 2>&1 | grep -i "access-control"
```

**Record the results clearly:**

- If Test 1 fails with CORS error: Use token-in-body approach (fallback)
- Record the JWT signing algorithm (HS256/ES256/RS256)

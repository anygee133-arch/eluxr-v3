**Part B: Design the campaign/themes two-step insert**

The v1 "Save Themes to Google Sheets" node appends flat rows. In v2, we need:
1. UPSERT into `campaigns` table (with user_id + month as unique constraint)
2. Capture the returned campaign_id
3. INSERT batch into `themes` table with that campaign_id FK

Design the exact node flow:
- Code node: Extract campaign metadata (month, user_id) from Claude response
- HTTP Request: UPSERT campaign -> returns campaign row with id
- Code node: Prepare themes array with campaign_id FK
- HTTP Request: INSERT batch themes

Read the "Claude -- Netflix Theme Generator" HTTP Request node and the "Parse & Split Theme Days" Code node to understand the exact data shape coming from Claude.
